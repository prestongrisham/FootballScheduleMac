App: Football Schedule

To Run Executable:
The executable file is FootballSchedule.app Ð on a Mac simply double clicking on the file will 
launch and run the app.

Compile:
To compile the code a Mac computer is needed running at least xcode 7+.  Double clicking the 
FootballSchedule.xcodeproj file will open all the related files in xcode and allow for building and 
running as a mac application.

Main Files:
The AppDelegate.swift file is the main call and mostly generic stating code from Apple.

The ViewController.swift file contains all the code to handle the data and display information to 
the user.
